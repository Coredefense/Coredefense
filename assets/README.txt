Guía de imágenes (assets):

- Coloca el logo de CORE DEFENSE como "logo.png" en esta carpeta.
  Tamaño recomendado: 256x256 px, fondo transparente.

- Coloca el banner principal como "banner.png" en esta carpeta.
  Proporción recomendada: 16:9 (ej. 1920x1080).
  La página lo usa como fondo del hero.

Si no se encuentran estas imágenes, el sitio mostrará el logo vectorial y un fondo degradado por defecto.
